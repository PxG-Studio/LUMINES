# WISSIL Developer Portal

Welcome to the WISSIL Developer Portal. This is your single source of truth for all WISSIL documentation.

## What is WISSIL?

WISSIL is a browser-based IDE for Unity development, featuring:

- **Ignis**: Visual Blueprint Editor (Node-based scripting)
- **Slate**: Design System
- **Ignition**: Runtime & Hot Reload
- **Spark**: Template System
- **Waypoint**: AI Assistant (LUNA)
- **Unity Tools**: Scene Graph, Prefabs, Shaders, Timeline, UI Canvas
