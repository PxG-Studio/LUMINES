# @lumines/ui

Shared UI components for LumenForge.io

## Purpose

Components used by both the main site (`apps/web`) and Storybook (`apps/storybook`):
- Panels
- Buttons
- Inputs
- 3-column layout
- MCP assistant panel
- SPARK loader components
- Storybook stories
